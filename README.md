## 说明

为vn.py提供了arctic的使用支持

## 安装

安装需要基于2.4.0版本以上的[VN Studio](https://www.vnpy.com)。

运行pandas版本需要低于1.0.3

运行python版本需要低于3.9

下载解压后在cmd运行：

```
python setup.py install
```
